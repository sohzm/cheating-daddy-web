Clear Storage

Reset all app data and settings.

This will remove all settings, saved configurations, and cached data. You'll need to reconfigure the app after.
1

Close Cheating Daddy

Make sure the app is completely closed before proceeding.
2

Delete the storage folder

Windows
macOS

Delete these folders:

C:\Users\<YourUsername>\AppData\Local\cheating-daddy
C:\Users\<YourUsername>\AppData\Roaming\cheating-daddy

Delete this folder:

~/Library/Application Support/cheating-daddy
3

Restart the app

Launch Cheating Daddy again. You'll need to reconfigure:
  • API key
  • App settings
  • Window position
